# Game Dev Portfolio

This is my GitHub Pages website showcasing my game development portfolio.

## About

A modern, responsive portfolio website designed to showcase game development projects, skills, and experience.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional design with smooth animations
- **Project Showcase**: Display your game development projects with descriptions and tags
- **Skills Section**: Highlight your technical skills and tools
- **Contact Section**: Easy ways for people to get in touch

## Structure

- `index.html` - Main portfolio page with all sections
- `styles.css` - Complete styling with responsive design
- `script.js` - Interactive features and smooth scrolling

## Sections

1. **Hero**: Eye-catching introduction
2. **About**: Brief introduction about yourself
3. **Projects**: Showcase your game development work
4. **Skills**: Display your technical abilities
5. **Contact**: Links to reach out

## Customization

To personalize this portfolio:

1. Update the content in `index.html` with your own information
2. Replace placeholder project descriptions with your actual projects
3. Add your own project images in the project cards
4. Update contact links with your actual email and social media
5. Modify colors in `styles.css` by changing the CSS variables

## GitHub Pages Setup

This site is ready to be deployed on GitHub Pages:

1. Go to your repository settings
2. Navigate to "Pages" section
3. Select the branch to deploy (usually `main` or `master`)
4. The site will be available at `https://yourusername.github.io/portfolio`
